Find-chinese
=========

Find chinese in your project code.

## Installation

  ```bash
  npm install -g find-chinese
  ```

## Usage

  ```bash
  find-chinese --path=wow/chrome/browser --suffix=["html","js","css"]
  ```
  
  The result will be in `chinese.txt`
